# LeetCode Solutions

A structured repository containing LeetCode solutions in Python, organized by topic and difficulty, along with helpful study notes.

## Directory Structure

- **[Arrays/](Arrays/)** - Array-based challenges (Two Sum, Merge Sorted Array, etc.)
- **[Strings/](Strings/)** - String manipulation and search algorithms.
- **[Linked_List/](Linked_List/)** - Singly and doubly linked list problems.
- **[Stack/](Stack/)** - LIFO problems, parentheses matching.
- **[Queue/](Queue/)** - FIFO problems.
- **[Hashing/](Hashing/)** - Key-value hashing, lookups, frequencies.
- **[Binary_Search/](Binary_Search/)** - Search-space reduction algorithms.
- **[Trees/](Trees/)** - Binary Tree traversals and structures.
- **[Graphs/](Graphs/)** - BFS, DFS, and topological sort.
- **[Dynamic_Programming/](Dynamic_Programming/)** - Overlapping subproblems.
- **[Notes/](Notes/)** - Conceptual study notes (Arrays, Linked List, Trees).

## How to Use
Navigate to any topic folder to find problem files. Each file contains a Python solution with time/space complexity details.
