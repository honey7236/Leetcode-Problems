# Linked List Study Notes

## Core Concepts
- Nodes containing value and a reference (pointer) to the next node.
- Non-contiguous memory allocation.
- O(1) insertion/deletion at pointers, O(N) random access.

## Common Patterns
1. **Dummy Node**: Simplifies edge cases (e.g., deleting head).
2. **Fast and Slow Pointers**: Detecting cycles, finding the middle element.
3. **Reverse Linked List**: Modifying pointers in-place.
