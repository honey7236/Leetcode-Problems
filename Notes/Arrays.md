# Array Study Notes

## Core Concepts
- Contiguous memory allocation.
- O(1) random access by index.
- O(N) insertion and deletion.

## Techniques
1. **Two Pointers**: Used in sorted arrays (e.g., Two Sum II).
2. **Sliding Window**: Used for subarray problems (e.g., maximum sum subarray).
3. **Prefix Sum**: Precomputing sums for range queries.
